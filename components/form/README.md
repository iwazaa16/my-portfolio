# お問い合わせフォーム

JavaScript によるクライアントサイドバリデーションを実装した、シンプルなお問い合わせフォームです。

## 概要

名前・メールアドレス・お問い合わせ内容を入力して送信できるフォームを実装しています。
入力内容を JavaScript でチェックし、条件を満たしていない場合はエラーを表示します。
全項目が正しく入力された場合のみ送信処理を行い、完了ページ（thanks.html）へ遷移します。

## 主な機能

* 入力フォーム（名前 / メールアドレス / お問い合わせ内容）
* JavaScript によるバリデーション

  * 名前・本文：空白を除いて入力されているかチェック
  * メールアドレス：正しいメール形式かチェック
* エラー表示

  * 不正な項目にエラーメッセージを表示
  * input イベントで条件を満たした瞬間にエラーを解除
* 送信制御

  * submit 時に `preventDefault()` でページ遷移を防止
  * 全項目 OK の場合のみ送信処理を実行
  * 送信中はボタンを無効化し、二重送信を防止
* 完了ページ遷移

  * 送信成功時に thanks.html へ遷移
  * sessionStorage を利用して直アクセスを防止

## 工夫した点

* **関数分割による可読性向上**
  バリデーション・エラー表示・UI制御をそれぞれ関数に分け、処理の役割が分かるようにしました。

* **boolean を使った判定の統一**
  バリデーション関数はすべて true / false を返すようにし、条件分岐をシンプルにしています。

* **リアルタイムバリデーション**
  submit 時だけでなく、入力中に条件を満たしたら即エラーを解除することで、ユーザー体験を意識しました。

* **実務を想定した完了ページ構成**
  アラート表示ではなく thanks.html を用意し、送信完了ページとして表示。
  また、sessionStorage を使って直接アクセスできないよう制御しています。

## 使用技術

* HTML
* CSS
* JavaScript（Vanilla JS）

## 備考

送信処理には `fetch` を使用し、テスト用に外部のダミー API（jsonplaceholder）へリクエストしています。
実際の運用ではバックエンドの API に差し替える想定です。
